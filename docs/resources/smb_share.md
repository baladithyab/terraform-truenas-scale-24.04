# Resource: truenas_smb_share

Manages SMB/CIFS shares on TrueNAS Scale, allowing Windows clients to access files and folders on the TrueNAS server.

## Example Usage

```terraform
resource "truenas_smb_share" "example" {
  name       = "myshare"
  path       = "/mnt/tank/mydata"
  comment    = "SMB share for mydata"
  enabled    = true
  browsable  = true
  guestok    = false
  readonly   = false
  recyclebin = true
  shadowcopy = true
}
```

## Import

Import is supported using the following syntax:

```shell
terraform import truenas_smb_share.example 1
```

### Required

- `name` (String) The name of the SMB share
- `path` (String) The path to share

### Optional

- `browsable` (Boolean) Whether this share is browsable. Defaults to `true`.
- `comment` (String) SMB share comment
- `enabled` (Boolean) Whether this share is enabled. Defaults to `true`.
